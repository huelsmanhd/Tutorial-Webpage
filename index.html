Functions

The purpose of a function is to allow us to preform an action or actions. Functions are everywhere in programming and essential to any webpage. A function can perform singular or repeated tasks. Functions are composed of a few different elements, such as the Declaration, Expression, and Parameters. Th declaration tells us what the variable(s) of a function will be. The expression shows what will be done with the variable.

Arrays

An array is a list of values. Unlike objects, arrays do not have corresponding keys. Data within an array is typically of the same type, and can be accessed via indexing.

Objects

Objects are similar to arrays, but they can hold multiple forms of information. The properties within an object can hold a range of varied, but related, information. Within an object is a key and a corresponding value. These values can be any type of data, including other objects.

For Loops

Loops are commonly used in coding to repeat a certain action until a specified condition is met. A For Loop consists of a few elements to set the conditions for the repeated action. The structure begins with the variable setup, followed by the test expression that typically sets a range of values to test, then the increment of each looped action completes the structure. On the following line, the test is run in the body of the loop until the parameters for the looped action are false.For Loops are very useful for actions that require repeated counting and listing.

Conditionals

A conditional is an essential piece of code syntax, providing a structure for true/false indicators. A conditional begins with the declaration of a variable to be compared to a true/false statment. The variable declaration is followed by an 'if' statement, that checks the variable against a certain condition. If that condition is met, the console.log function runs an input phrase that expresses the variable as true.
